# Sparq design references

Static HTML mockups from the Sparq design system, sourced from Sparq.zip dated 2026-04-27. Reference only — not part of the production build.

- `sparq-tokens.css` is the source of truth for design tokens. Its values have been ported into `tailwind.config.ts` and `src/app/globals.css`.
- `sparq-workspace.css` contains layout helpers used only by these mockup pages; do not import it into the app.
- View locally at http://localhost:3000/design/index.html during dev.
